# Padrões de Projeto de Software
- Repositório dedicado aos meus estudos, trabalhos e projetos envolvidos na disciplina de Padrões de Projeto de Software (*PPS*).

## *Sketchs* no Excalidraw
- O repositório conta com algumas *sketchs* (rascunhos) desenhados que faço durante as aulas para auxiliar meus estudos. Caso deseje vê-los, siga estas instruções:

    1. Localize o padrão que deseja através do nome da pasta;
    2. Procure um arquivo chamado *OpenInExcalidraw* que possua a extensão `.excalidraw`;
    3. Baixe esse arquivo e vá até o site [Excalidraw](https://excalidraw.com/);
    4. Abra o arquivo no site e você conseguirá visualizar os desenhos.
